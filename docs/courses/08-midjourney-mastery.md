# Module 8 — Midjourney Mastery

**Code path:** `src/data/content/modules/08-tools-midjourney/`

## Purpose

This module addresses visual AI literacy — a distinct skill set from text-based AI interaction. It teaches image generation through prompt construction, parameter control, style consistency, and commercial usage considerations. No prior design training is assumed.

## Curriculum Position

Module 8 is the only primarily non-text module in the catalog. It is positioned after the text-focused tool modules (Modules 5–7) so that learners approach visual generation with established prompting discipline. It also pairs with Module 9 (Workflow and Automation), where image generation can be integrated into multi-step professional workflows.

## Pedagogical Approach

The module follows the same mastery pattern as other tool modules: foundations, prompt craft and parameters, applied use cases (business and personal), advanced consistency techniques, and commercial rights awareness. The Discord-based interface is addressed explicitly in early lessons to reduce friction from the unfamiliar tool surface.

## Structure

- 8 chapters, 27 lessons, approximately 405 minutes total estimated duration.
- Chapter progression: Getting Started, Prompt Fundamentals, Parameters, Business Applications, Personal Applications, Advanced Consistency, Refinement Workflows, Commercial and Rights.
