# Curriculum

This directory contains rationale documentation for each of the 10 curriculum modules. Each file describes the pedagogical purpose, curriculum position, and structural composition of its module. The curriculum design is discussed in Thesis §3.6.

## Module Sequence

The 10 modules follow a deliberate progression:

- **Modules 1–4** establish foundational understanding and transferable skills (AI literacy, productivity, prompt engineering, content creation).
- **Modules 5–8** provide tool-specific depth for four major AI platforms (ChatGPT, Claude, Gemini, Midjourney).
- **Module 9** integrates multiple tools into coherent multi-step workflows.
- **Module 10** applies accumulated skills through a structured 28-day daily challenge.

The personalization algorithm (Thesis §3.7.3) selects and ranks modules from this catalog based on each learner's onboarding responses. Learners are not required to follow the numerical sequence.
