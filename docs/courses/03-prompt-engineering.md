# Module 3 — Prompt Engineering

**Code path:** `src/data/content/modules/03-skills-prompting/`

## Purpose

This module makes the skill of communicating with AI models explicit and systematic. It covers prompt structure (role, task, format, examples), chain-of-thought guidance, iterative refinement, and troubleshooting when outputs are unsatisfactory. Prompt engineering is a transferable skill that benefits every tool-specific module that follows.

## Curriculum Position

Module 3 is positioned after productivity (Module 2) so that learners approach prompting with motivation grounded in real tasks. It precedes content creation (Module 4) and all tool-specific modules (Modules 5–8), serving as the shared literacy layer for effective interaction with any AI model.

The personalization algorithm frequently recommends this module as the first or second module for intermediate users, as indicated by the rank-1 distribution data in the algorithm audit (Thesis §4.2).

## Pedagogical Approach

Lessons prioritize practical habit formation over terminology. Learners build prompts incrementally, practice iteration across multiple rounds, and assemble a personal prompt library by the end of the module. Troubleshooting — identifying why an output missed and applying concrete fixes — is treated as a core skill.

## Structure

- 1 chapter, 10 lessons, approximately 120 minutes total estimated duration.
- Learning outcomes: structured prompt construction, context management, iterative refinement, reusable template library.
