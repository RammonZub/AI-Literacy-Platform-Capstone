# Module 5 — ChatGPT Mastery

**Code path:** `src/data/content/modules/05-tools-chatgpt/`

## Purpose

This module provides comprehensive, product-specific instruction for ChatGPT. It covers the full range of the platform's capabilities — from essential conversational interactions through professional application workflows, advanced features, and third-party integrations — structured as a progressive learning path.

## Curriculum Position

Module 5 is the first tool-specific deep-dive module in the catalog, following the vendor-neutral skill modules (Modules 1–4). It establishes the structural pattern that subsequent tool modules (Claude, Gemini, Midjourney) follow: essentials, applied workflows, advanced features, and a capstone integration exercise.

## Pedagogical Approach

The module balances breadth and depth. Earlier chapters cover essential interactions and professional applications. Later chapters introduce advanced features (file handling, plugins, custom instructions) and conclude with integrations that connect ChatGPT to broader professional workflows. Each chapter ends with an applied exercise grounded in a realistic professional scenario.

## Structure

- 5 chapters, 30 lessons, approximately 10 hours total estimated duration.
- Chapter progression: Essentials, Professional Applications, Advanced Features, Advanced Deep Dive, Integrations and Capstone.
