# Module 7 — Gemini Mastery

**Code path:** `src/data/content/modules/07-tools-gemini/`

## Purpose

This module covers Google's Gemini with emphasis on its integration with the Google Workspace ecosystem (Docs, Sheets, Gmail), real-time web grounding capabilities, and mobile-first usage patterns. It is designed for professionals whose daily workflows are already centered on Google products.

## Curriculum Position

Module 7 is the third tool-specific depth module. It assumes general prompting competence from Module 3 and complements the ChatGPT and Claude modules by addressing contexts where live information retrieval and Google-native integrations are the primary requirements.

## Pedagogical Approach

The module progresses from prompting fundamentals through Workspace-specific applications, mobile and real-time interaction patterns, professional use cases, and responsible use considerations. Mobile scenarios are treated explicitly, as the risk profile and interaction patterns differ from desktop-only text generation.

- 8 chapters, 27 lessons, approximately 405 minutes total estimated duration.
- Chapter progression: Essentials, Prompting, Workspace Integration, Mobile, Real-Time Features, Professional Applications, Privacy and Responsible Use, Future-Building.
