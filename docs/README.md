# Documentation

This directory contains implementation documentation for the AI Literacy Platform. Each subdirectory addresses a distinct aspect of the system: architecture and design decisions, the learner-facing platform, the curriculum structure, application configuration, and pilot study evidence.

## Contents

| Directory | Purpose |
| ---- | ---- |
| [architecture/](./architecture/README.md) | System architecture, personalization algorithm, data model, and content structure. |
| [platform/](./platform/README.md) | Learner platform flow: onboarding, dashboard, lesson delivery, and progress tracking. |
| [courses/](./courses/README.md) | Curriculum rationale for each of the 10 modules and their pedagogical sequencing. |
| [app-settings/](./app-settings/README.md) | Authentication, telemetry, and access control configuration. |
| [pilot-study/](./pilot-study/README.md) | Pilot study participant data and results. |
| [figures/](./figures/) | Generated figures for thesis Chapter 4 (algorithm distribution, SUS scores, activation funnel). |
